🛠️ 설치 가이드 🧡 필독 🧡

처음 설치하시는 분은 아래 순서를 하나씩 따라해 주세요.

1
zip 파일 다운로드 → 압축 풀기
위 다운로드 링크에서 uto-grok-v2.0.0.zip 파일을 받고, 압축을 풀어주세요.
2
압축 푼 폴더 확인
폴더 안에 파일들이 보이면 정상입니다. 건드리지 마시고 다음 단계로 넘어가세요!
3
Chrome에서 확장프로그램 페이지 접속
Chrome 주소창에 chrome://extensions 를 입력하고 엔터를 눌러주세요.
4
개발자 모드 켜기
우측 상단의 "개발자 모드" 토글을 켜주세요. 3가지 버튼이 새로 나타납니다.
5
"압축해제된 확장 프로그램을 로드" 클릭
나타난 3가지 버튼 중 "압축해제된 확장 프로그램을 로드합니다" 를 클릭하세요.
6
우토그록 폴더 선택
우토그록 압축 푼 폴더에 들어가서 아무것도 건드리지 말고,
압축 푼 폴더를 1번 클릭 (더블클릭 X) 하고 "폴더 선택" 버튼을 눌러주세요.
⚠️zip 파일을 그대로 로드하면 안 됩니다! 반드시 압축을 먼저 풀어주세요.
7
설치 완료!
확장프로그램 목록에 우토그록이 나타나면 설치 완료입니다.
8
grok.com 접속 → 우토그록 아이콘 클릭
grok.com/imagine 에 접속한 후, Chrome 툴바에서 우토그록 아이콘을 클릭하면 사이드패널이 열립니다.

🔍 업스케일 기능

품질 설정에서 480p-업스케일 또는 720p를 선택하면, 동영상 생성 완료 후 자동으로 업스케일이 진행됩니다.

동영상 생성 → 추가 옵션 메뉴 열기 → "동영상 업스케일" 클릭 → 업스케일 완료 대기 → 다운로드

📊 완료 요약 및 재실행

작업이 모두 완료되면 결과 요약이 표시됩니다:

✅ 완료! 성공 15개 / ❌ 실패 2개 / 🔍 업스케일 13개

실패한 항목이 있으면 "🔄 실패 N건 재실행" 버튼이 자동으로 나타납니다. 클릭하면 실패한 것만 다시 실행합니다.

⚠️ 주의사항

  • grok.com 계정에 로그인된 상태여야 합니다
  • grok.com의 Imagine 페이지에서만 작동합니다
  • 너무 빠르게 연속 생성하면 rate limit (429/409 에러)이 걸릴 수 있습니다. 딜레이를 30초 이상으로 설정하세요
  • grok.com UI가 업데이트되면 일부 기능이 작동하지 않을 수 있습니다. 이 경우 확장프로그램 업데이트를 기다려주세요

문제가 발생했을 때 아래 순서대로 확인하세요. 오류나면 무조건 여기 먼저 보세요!

1단계 기본 확인

  • ✅ grok.com에 로그인 되어 있나요?
  • ✅ grok.com/imagine 페이지에 있나요?
  • ✅ 확장프로그램이 최신 버전인가요?
👉하나라도 아니면 → 로그인 / Imagine 이동 / 최신 버전 다운로드

2단계 실행 중 에러

"전송 버튼을 찾을 수 없습니다"
→ 🔃 새로고침 버튼 클릭 → 2~3초 대기 → 다시 실행
→ 계속 실패 시 딜레이를 60초 이상으로 변경
"Could not establish connection"
→ 🔃 새로고침 버튼 클릭 → 2~3초 대기 → 다시 실행
"409 Conflict" 반복
→ rate limit 상태 → 5~10분 대기 후 다시 시작
→ 딜레이를 60초 이상으로 올리기

3단계 업스케일 안 됨

→ 설정에서 품질이 480p-업스케일 또는 720p인지 확인
→ 맞는데도 안 되면 → 🔃 새로고침 후 다시 실행
→ 최신 버전이 아니라면 → 최신 버전으로 업데이트

4단계 그래도 안 될 때

  1. grok.com에서 수동으로 동영상 1개 생성해보기
  2. 수동 생성도 안 되면 → grok.com 자체 문제 (계정 한도 초과 등)
  3. 수동은 되는데 확장프로그램만 안 되면 → 🐛 버그신고 → 카페 게시글에 댓글

❓ FAQ 자주 묻는 질문

최소화 해도 우토그록 진행 되나요?▼

아래 방법은 작업이 계속 진행됩니다:

✅ 다른 탭으로 전환 → 돌아감!
✅ 창을 뒤로 보내기 (다른 창 위에 올리기) → 돌아감!
✅ 창 크기를 작게 줄여서 구석에 놓기 → 돌아감!

⚠️ 창을 최소화하면 멈출 수 있습니다.
대신 다른 탭으로 전환하거나, 창을 작게 줄여서 구석에 놓으세요!

확장프로그램을 설치했는데 아이콘이 안 보여요▼

Chrome 툴바의 퍼즐 아이콘(🧩)을 클릭한 뒤, 우토그록 옆의 📌 핀 버튼을 눌러주세요.

사이드패널을 열었는데 "Grok 페이지가 필요합니다" 팝업이 떠요▼

grok.com이 아닌 다른 사이트에서 열었을 때 나타납니다. "Grok으로 이동" 버튼을 누르면 grok.com/imagine 페이지로 바로 이동합니다.

"전송 버튼을 찾을 수 없습니다" 에러가 나와요▼

이미지 업로드가 grok.com에 정상적으로 인식되지 않았을 때 발생합니다. 주요 원인:

1. rate limit — 너무 빠르게 연속 작업해서 이미지 업로드가 거부됨
2. 페이지 상태 꼬임 — 이전 작업의 잔여물이 남아있음
3. grok.com UI 변경 — 업로드 영역 구조가 바뀜

→ 하단의 🔃 새로고침 버튼을 누른 뒤 다시 실행해보세요. 계속 실패하면 딜레이를 60초 이상으로 올려주세요.

409 Conflict 에러가 계속 나와요▼

grok.com의 rate limit에 걸린 것입니다. 딜레이 설정을 60초 이상으로 올려보세요. 이미 rate limit에 걸렸다면 5~10분 정도 기다린 후 다시 시작하세요.

"Could not establish connection" 에러가 떠요▼

grok.com 페이지에서 content script가 아직 로드되지 않았거나, 새로고침 직후에 발생합니다. 🔃 새로고침 버튼을 누른 뒤 2~3초 기다렸다가 다시 실행하세요.

업스케일이 안 돼요 / 건너뛰기됩니다▼

1. 설정에서 품질이 480p-업스케일 또는 720p로 되어있는지 확인하세요
2. 동영상 생성 직후 화면에 "추가 옵션" 버튼(⋯)이 보이는지 확인하세요
3. 최신 버전이 아니라면 업데이트하세요

중간에 멈추고 싶어요▼

Esc 키를 누르거나, 사이드패널의 ■ 중지 버튼을 클릭하세요. 현재 진행 중인 작업이 완료된 후 멈춥니다.

grok.com 업데이트 후 작동이 안 돼요▼

grok.com의 UI 구조(DOM)가 바뀌면 확장프로그램이 작동하지 않을 수 있습니다. 카페 게시글에 댓글로 알려주시면 최대한 빠르게 반영하겠습니다.

콘솔에 "user-skills 403 Forbidden"이 뜹니다▼

grok.com 자체 에러로 우토그록과 무관합니다. grok.com 내부 API 호출이 실패하는 것이며, 동영상 생성에는 영향 없으니 무시하셔도 됩니다.
